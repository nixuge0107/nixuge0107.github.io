响应内容

  • 对于文本请求，使用r.text获得。
    通常Requests 会自动解码来自服务器的内容。大多数 unicode 字符集都能被无缝地解码。
    若解码出现乱码，使用r.encoding进行修改使用的编码方式。
  • 对于图片等内容，使用r.content进行请求。
  • 对于JSON内容，使用r.json()进行请求。
